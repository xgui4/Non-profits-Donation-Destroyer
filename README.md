# Wikipedia Donation Destroyer

This is a simple browser extension which does 3 things (and 3 things only)

1. Blocks the Wikipedia "Donate or Wikipedia will die in 3 minutes!" popups.
2. Displays the true financials of Wikipedia and others similar sites in a small, closable box.
3. Provides a "See if this page is available on Grokipedia" link (for wikipedia articles only).

The "Not-profits" Donation Destroyer will also display financial data for other Tech Foundations if you are on their websites.

## How to Install

### For Google Chrome / Chromium / Brave / Edge

1. Download the extension source files.
<img width="965" height="133" alt="Chrome Developper Mode" src="https://github.com/user-attachments/assets/4247cb8e-f671-4e32-bf4e-9021bc77fb27" />
2. Within Chrome, go to `chrome://extensions/` and enable **"Developer mode"**.
3. Click **"Load unpacked"** and select the extension folder.

Boom!  Done!  If you don't like the extension, you can remove or disable it from [chrome://extensions/](chrome://extensions/).

### For Firefox

> [!NOTE]
> This will only work for the current session, unless you sign the addon with a `.xpi` extension, the addon will be temporaly.
> Unless you do `xpinstall.signatures.required == false` in about:config, but this might only work on Dev Edition and Nightly
> This is due to an limitation by mozilla and not this addon

1. Download the extension source files.
2. Within Firefox, go to `about:debugging#/runtime/this-firefox`.
